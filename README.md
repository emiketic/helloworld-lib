# Hello World Library

![](https://img.shields.io/david/emiketic/helloworld-lib.svg?style=for-the-badge)

A collection of common utilities.

## Modules

- [`common`](./src/common)

## Usage

```sh
# install dependencies
npm install

# lint code for critical issues
npm run lint:critical

# lint code
npm run lint

# format code
npm run format

# run tests
npm run test

# run build
npm run build
```
